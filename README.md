# Los-Pollos-Hermanos
"Project Description: (1) 
Develop a web-based application for a private hospital using C# and ASP.NET. The system should 
include core functionalities such as patient registration, doctor management, appointment 
scheduling, and medical record management, with appropriate data validation applied throughout. 
The application must feature a user-friendly interface built with Bootstrap, ensure data integrity, and 
showcase key ASP.NET capabilities, including server controls and SQL Server database connectivity."
